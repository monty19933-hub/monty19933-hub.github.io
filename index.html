Forecast cards use computer model guidance from Open-Meteo. The forecast uses HRRR/GFS, NAM, and NBM for days 1-2, then NBM only for days 3-9. The NWS feed is used for current conditions and active watches, warnings, advisories, and statements.

Potential record high/low coloring compares the forecast temperature against Open-Meteo historical reanalysis daily extremes from 1991 through last year for the same point location and calendar date.

Severe Chance is a model-derived proxy from thunderstorm probability, CAPE, wind gusts, and rain amount; it is not an official severe weather outlook.

City and ZIP search uses geocoding only to find latitude and longitude; forecast cards still use the model guidance described above.

Icons use the local 98-06 TWC icon pack you provided, with motion added only for rain, snow, thunderstorm, and wind states.
